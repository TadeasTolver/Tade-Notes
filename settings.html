Notes from the developer

Note storage method

Notes are purely stored using local storage. Tade Notes are designed to be used for simple note taking, it shall not be used to store important information, as it indisputably can be lost by accident, foolhardy users, or technical issues.

Preventing data loss

To prevent data loss, do not command localStorage.clear() in your browser console, as it will delete all your notes irreversebly. If you have any queries, please contact the developer:
